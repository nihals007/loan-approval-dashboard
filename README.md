# 🚀 Loan Approval ML Pipeline Dashboard

## 📌 Overview

This project is an **end-to-end Machine Learning pipeline dashboard** built using Streamlit. It allows users to upload datasets, perform EDA, preprocess data, train models, evaluate performance, and make real-time predictions.

---

## 🧠 Features

* 📊 Exploratory Data Analysis (EDA)
* 🧹 Data Cleaning & Feature Engineering
* 🎯 Feature Selection (Random Forest Importance)
* 🤖 Model Training (Logistic Regression / others)
* 🔁 K-Fold Cross Validation
* 📈 Performance Metrics (Accuracy, Precision, Recall, F1, ROC-AUC)
* 🔍 Confusion Matrix Visualization
* 🚀 Live Loan Prediction Interface

---

## 🏗️ Pipeline Steps

1. Input Data
2. EDA
3. Data Cleaning
4. Feature Engineering
5. Feature Selection
6. Data Split
7. Model Training
8. K-Fold Validation
9. Performance Evaluation
10. Live Prediction
